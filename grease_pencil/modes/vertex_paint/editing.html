Editing¶

Set Vertex Colors¶

Reference

Mode

Vertex Paint Mode

Menu

Paint ‣ Set Vertex Colors

Sets the active color to all selected vertices.

Reset Vertex Colors¶

Reference

Mode

Vertex Paint Mode

Menu

Paint ‣ Reset Vertex Colors

Removes vertex color information of the active strokes, if no strokes are selected, all strokes are reset.

Invert¶

Reference

Mode

Vertex Paint Mode

Menu

Paint ‣ Invert

Invert RGB values.

Menu
